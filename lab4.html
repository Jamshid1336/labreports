Fractal Tree

There are many criteria in this code that can be changed, such as the length, the depth, the degrees, the color, and the amount by which the length decreases each time the function runs. By increasing the depth, the face direction, the amount of branches is increased, but each continuing branch becomes shorter and so the tree becomes very dense. Alternately, the degrees that the branches turn can be changed, making the tree spread out more or become narrow. 



def fractal_tree(length, depth):
    if depth == 0:
        return
    else:

        forward(length)
        left(15)

        fractal_tree(length * 0.7, depth - 1)

        right(30)

        fractal_tree(length * 0.7, depth - 1)

        left(15)
        backward(length)
